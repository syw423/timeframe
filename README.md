# TimeFrame — 3D 时光相册

> 将你的照片文件夹变成一座可漫游的 3D 记忆空间。

## 特性一览

| 功能 | 说明 |
|------|------|
| 多种视图 | 网格 / 轮播 / 3D 环形 / 翻页书 / 3D 手势，一键切换 |
| 日期筛选 | 底部日历弹窗，有照片的日期自动高亮 |
| 双页翻书 | 像真相册一样左右翻页，照片完整显示带留白 |
| 本地缓存 | IndexedDB 自动存储，刷新页面不用重新导入 |
| 备注支持 | 每个日期文件夹放 `note.txt`，查看照片时自动显示 |
| 深色主题 | 太空感深色背景 + 毛玻璃效果，护眼又好看 |

## 快速开始

### 方式一：双击启动（Windows）

```bash
start-server.bat
```

浏览器自动打开 `http://localhost:8000`

### 方式二：命令行

```bash
python -m http.server 8000
# 或
npx http-server -p 8000
```

然后打开 `http://localhost:8000`

## 使用指南

### 1. 准备照片

把你的照片按日期放到文件夹里：

```
我的照片/
├── 2024-06-01/
│   ├── 01_海边.jpg
│   ├── 02_沙滩.jpg
│   └── note.txt      ← 可选：该日期的备注
├── 2024-06-15/
│   ├── 01_草坪.jpg
│   └── 02_花朵.jpg
└── 2025-01-01/
    ├── 01_烟花.jpg
    └── 02_晚餐.jpg
```

文件夹名就是日期（`YYYY-MM-DD`），程序会自动识别。

### 2. 导入照片

点击「导入照片」→ 选择文件夹 → 照片自动加载。

### 3. 切换视图

顶部工具栏有 4 个视图按钮：

- **网格** — 平铺展示，适合快速浏览
- **轮播** — 单张大图滑动，适合聚焦观看
- **3D 环形** — 照片环绕成圆环，可拖拽旋转
- **翻页书** — 双页展开像真相册，左右翻页

### 4. 日期筛选

点击「选择日期」→ 日历弹窗中高亮的日期就是有照片的日期 → 点击跳转。

### 5. 查看详情

双击任意照片 → 弹出详情面板，显示大图、日期、备注、文件名。

### 6. 添加更多照片

导入一次后，点击「添加照片」可以继续导入新文件夹，不会覆盖已有照片。

## 项目结构

```
timeframe/
├── index.html          # 入口页面
├── style.css           # 全部样式（深色主题 + 毛玻璃）
├── script.js           # 主逻辑：导入、缓存、视图切换
├── start-server.bat    # Windows 一键启动脚本
├── src/
│   ├── book-layout.js      # 翻页书布局
│   ├── carousel-layout.js  # 轮播布局
│   ├── grid-layout.js      # 网格布局
│   ├── three-mode.js       # 3D 环形/手势（Three.js）
│   ├── date-utils.js       # 日期工具函数
│   └── db.js               # IndexedDB 缓存
├── example/            # 示例照片文件夹（见下方）
└── screenshots/          # 项目截图
```

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + ES6+
- **3D 渲染**: Three.js（环形/手势模式）
- **3D 翻书**: CSS 3D Transforms
- **缓存**: IndexedDB + 原生 File API
- **EXIF 读取**: exif-js

```bash
python -m http.server 8000

# 3. 打开浏览器
http://localhost:8000
```

## 注意事项

- **必须用本地服务器打开**：直接双击 `index.html` 会导致文件 API 权限不足，无法选择文件夹。
- **照片保留在本地**：所有照片只存在你的浏览器里，不会上传到任何服务器。
- **首次导入后自动缓存**：刷新页面后照片会从 IndexedDB 恢复，无需重新导入。
